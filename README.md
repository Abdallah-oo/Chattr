## рҹ“ұ Overview

**Chattr** is a full-featured real-time messaging app built with Flutter. It supports private one-on-one conversations, group chats, voice messages, and image sharing вҖ” all backed by Supabase for real-time sync and offline-first caching powered by Hive.

## вңЁ Features

- рҹ’¬ **Private Chats** вҖ” Real-time one-on-one messaging with message status tracking (sent, delivered, read)
- рҹ‘Ҙ **Group Chats** вҖ” Create and manage group conversations with multiple participants
- рҹҺҷпёҸ **Voice Messages** вҖ” Record and send audio messages with live recording duration display
- рҹ–јпёҸ **Image Sharing** вҖ” Send images in chat with preview and download support
- рҹ“І **Contacts** вҖ” Browse and start conversations with your contacts
- вҡЎ **Offline-First** вҖ” Local caching with Hive for instant load and offline access
- рҹҢҷ **Dark Theme** вҖ” Sleek dark UI with a modern frosted-glass navigation bar
- рҹ”„ **Real-time Sync** вҖ” Supabase Realtime for live message updates across devices

## рҹҸ—пёҸ Architecture

Chattr follows a **Clean Architecture** approach with a feature-first folder structure.

```
lib/
в”ңв”Җв”Җ core/
в”Ӯ   в”ңв”Җв”Җ cubits/          # Shared cubits (audio, image picker, download...)
в”Ӯ   в”ңв”Җв”Җ routing/         # App-wide routing (go_router)
в”Ӯ   в”ңв”Җв”Җ services/
в”Ӯ   в”Ӯ   в”ңв”Җв”Җ hive/        # Local persistence (Hive boxes)
в”Ӯ   в”Ӯ   в””в”Җв”Җ supabase/    # Supabase client, storage, and realtime
в”Ӯ   в”ңв”Җв”Җ themes/          # App colors and theme config
в”Ӯ   в””в”Җв”Җ utils/
в”Ӯ       в”ңв”Җв”Җ di/          # Dependency injection (get_it)
в”Ӯ       в””в”Җв”Җ cache/       # In-memory user cache (UsersCache)
в”Ӯ
в””в”Җв”Җ features/
    в”ңв”Җв”Җ auth/            # Authentication (login, register)
    в”ңв”Җв”Җ contacts/        # Contacts list
    в”ңв”Җв”Җ private_chats/   # 1-on-1 chat feature
    в””в”Җв”Җ group_chats/     # Group chat feature
```

Each feature follows this internal structure:

```
feature/
в”ңв”Җв”Җ data/
в”Ӯ   в”ңв”Җв”Җ models/          # Hive-annotated data models
в”Ӯ   в””в”Җв”Җ repos/           # Data repository implementations
в”ңв”Җв”Җ domain/
в”Ӯ   в””в”Җв”Җ repos/           # Abstract repository contracts
в””в”Җв”Җ presentation/
    в”ңв”Җв”Җ cubits/          # Feature-level state management
    в””в”Җв”Җ views/           # Screens and widgets
```
